# Fci_Louts

الموقع الرسمي (Static) لكلية الحاسبات والمعلومات والذكاء الاصطناعي - جامعة اللوتس.

موقع HTML/CSS/JS ثابت، يدعم العربية (RTL)، ومتجاوب مع كل الأجهزة، مع تركيز خاص على وحدة ضمان الجودة.

## التشغيل
افتح `index.html` في المتصفح. كل صفحة مستقلة بذاتها (الـ CSS و JS مدموجان داخلها).

## الصفحات
- index.html — الرئيسية
- about.html — عن الكلية
- programs.html — البرامج الأكاديمية
- admission.html — القبول والتسجيل
- students.html — الطلاب
- faculty.html — أعضاء هيئة التدريس
- quality.html — وحدة ضمان الجودة
- research.html — البحث العلمي
- news.html / events.html / activities.html — الأخبار والفعاليات والأنشطة
- contact.html — اتصل بنا
